This Plugin fixes the SEO relevant duplicate content issue for FeedWordPress plugin.

== Description ==

If you aggregate content from another website using 
[FeedWordPress](http://wordpress.org/extend/plugins/feedwordpress/) and you
opt for linking the aggregated post to the local copy on your website you 
most likely run into the [duplicate content issue](http://support.google.com/webmasters/bin/answer.py?hl=en&answer=66359).

This plugin fixes this SEO relevant issue by setting the **rel="canonical"** 
meta tag to the original source.

== Installation ==

Upload the plugin to your blog and activate it as any other plugin. **It will 
work out of the box**.
